여기에 최애 사진(PNG)을 넣으면 이모지 대신 그 사진이 캐릭터로 나옵니다.

[방법 1] 상태별로 다른 사진 쓰기 (권장)
  coding.png    - 코딩 중일 때
  thinking.png  - 생각 중일 때
  done.png      - 다 끝냈을 때 (검토 요청)
  waiting.png   - 확인/허락이 필요할 때
  idle.png      - 대기 중일 때

[방법 2] 한 장으로 다 쓰기
  character.png  - 모든 상태에서 이 사진 하나만 사용
  (상태별 png가 있으면 그게 우선)

- 배경이 투명한 PNG면 더 예쁘게 나옵니다.
- 가로 150px 기준으로 표시됩니다.
- 파일을 넣거나 바꾸면 다음 상태 변경 때 자동 반영됩니다.
